Overview

This app is intended for use as a supplemental revision aid in your preparation for the Science exam. It should be used alongside the practice questions provided in the training notes and in consultation with your Tutor.

Instructions For Use

Choose a button from the left column for the subject you wish to revise. Each click will create a new sum. Once you have completed your answer on paper, click the ‘solution’ button to see a full, worked solution. A counter within the Solution button shows the number of times the solution has been viewed.

Link To Training Notes

Under each question, a link is provided which will take you to the relevant area of the training notes.

Vectors

Produces various different vector drawing exercises. In each case, use the specified type of scale drawing to solve the problem. Remember, these questions must be solved by scale drawing, not trigonometry. While the 'solution' is visible, a 'show me how' button will also be available which will run an animation showing how the diagram is constructed.

Moments / CofG

This button randomly produces either a question requiring you to find the centre of gravity of an aircraft or a pivoted beam to be put into a state of equilibrium.

All other buttons

Each button produces various problems on the related topic. In each case, the 'solution' button gives a worked solution.
